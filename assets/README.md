# LDCM Demo Assets

These samples are copied from `SpatialVision/Prior-Depth-Anything` assets and
prepared for LDCM demos.

- `image.png`: RGB input image.
- `sparse_depth.npy`: sparse prior depth in meters, `float32`, with missing points as `0`.
- `sparse_depth_mm_uint16.png`: same sparse prior, stored as uint16 millimeters.
- `gt_depth.npy`: ground-truth depth in meters, `float32`.
- `gt_depth_mm_uint16.png`: same ground truth, stored as uint16 millimeters.

Samples included: `sample_1`, `sample_2`, `sample_3`, `sample_5`.
